# Portfolio — Peniel Mabanza

Portfolio personnel de **Peniel Mabanza**, développeur full stack. Présente mon parcours, mes compétences et mes projets (BEPAS, Aksantiship, applications métiers, etc.).

## Stack

- **Next.js 16** (App Router)
- **React 19** + **TypeScript**
- **Tailwind CSS 4**

## Démarrage

```bash
npm install
npm run dev
```

Ouvre [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
npm run dev    # développement
npm run build  # build production
npm run start  # serveur production
npm run lint   # eslint
```

## Structure

```
app/           # layout, page, styles globaux
components/    # sections du portfolio
lib/           # données (projets, contact, compétences)
public/        # images, vidéos
```

## Assets projets

Remplace les mockups SVG dans `public/images/projects/` par de vraies captures PNG/WebP quand tu les as.
